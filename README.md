# Brute-Force-Uncipher
## Description
A little algorithm to uncipher a ciphered text with a substitution cipher algorithm, using a brute force method.

## Compiling
> make

## Running
> ./bin/uncipher 'pathToCipheredFile'

## Cleaning the project
> make clean
